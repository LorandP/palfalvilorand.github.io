Biblio

  I worked in C# (WFA), MySQL and used XAMPP and the project requires a relational database and a program for a library that could be used by administrators, librarians and users and everyone of them having their specific rights and access to specific features of the program.
The program used a login and sign-up feature and after authenticating the logged in user, according to its rights, granted access to Insert, Update or Delete into and from the database. For debugging I have run several unit, integration, system and smoke tests.
  I update this project whenever I learn new things that are useful and bring something new to the project like implementing a logout feature and giving it a more modern look and feel.

My game called: Shinigami

  After succesfully finishing the course offered by Brinel, I have continued learning and I startde a small project by developing a game in console application. This game has 10 levels and at each level you encounter stronger characters and I have stored these characters in a text file and I have the program load them from there after you succesfully advanced to the next level.
  You have 4 types of actions that you repeat while you defeat or have have been defeated by the character at that level. After each action, you and the character acumulate magic points, when full, the characters damage increasez for one attack, but you have 2 options eather to heal or to deal more damage. Also this game has an age restriction and it is still in continuous development.
